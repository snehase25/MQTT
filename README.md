**Tools and technologies used**
   
   React, 
   TypeScript,
   GoJS,
   Vite,
   Yarn

**Setup and Installation:**

   1) Make sure you have latest node installed, used version 18 to build
   2) clone the project: git clone https://github.com/snehase25/MQTT.git
   3) yarn intall
   4) yarn run dev
